# Eureka Project 🛒

> 기본적인 쇼핑기능을 제공하는 이커머스 웹사이트
>
> 어디서나 고객이 원하는 상품을 찾을 수 있도록 하는 Eureka의 핵심인 떠 다니는 카테고리
>
> 상품 검색, 주문, 리뷰, 판매자 상품등록, 관리자 페이지, 카트, 회원 관리 등의 기능을 지원하는 이커머스 웹사이트
